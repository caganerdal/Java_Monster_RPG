# Java_Monster_RPG

It's a game that I developed using Java.

**Characters:** Different classes like Savasci, Buyucu.

**Dynamic Combat System:** Randomized monster encounters with logic-based damage calculation.

**Object-Oriented Programming (OOP):** Deep dive into Inheritance, Polymorphism, and Encapsulation.

**Data Structures:** Efficient use of Arrays for inventory and entity management.
